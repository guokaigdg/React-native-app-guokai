> 1. 参数传递: navigate接受可选的第二个参数，以便将参数传递给要导航到的路由。例如：this.props.navigation.navigate('RouteName', {paramName:  'value'})。<br>

>3. 也可以使用 this.props.navigation.state.params作为getParam的替代方案， 如果未指定参数，它的值是 null。
